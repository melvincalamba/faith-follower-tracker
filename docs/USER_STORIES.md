# User Stories
## Ito ang format: "As a [user], I want to [action], so that [reason]."

## Users Description

* Admin and Mentor have the same base functionality. The only difference is that Admin has additional user management capabilities.

### Admin

* As an Admin, I want to add new members so that I can add my own follow-up (members).
* As an Admin, I want to view all members and their progress so that I can see the overall discipleship pipeline.
* As an Admin, I want to see all pending user registrations so that I can review who wants to join the system.
* As an Admin, I want to approve a pending user so that they can start logging in and using the system.
* As an Admin, I want to reject/delete a pending user so that I can prevent unauthorized access.
* As an Admin, I want to see all active users so that I know who currently has access to the system.
* As an Admin, I want to delete an active user so that I can remove someone who should no longer have access.
* As an Admin, I want to edit any member regardless of who their mentor is so that I can correct data errors.
* As an Admin, I want to delete any member regardless of who their mentor is so that I have full control over the data.

### Mentor / Users

* As a Mentor, I want to view all members in the system so that I have visibility of the entire discipleship pipeline.
* As a Mentor, I want to add, update, delete, and view my follow-up member so that I can start monitor their discipleship journey.
* As a Mentor, I want the system to automatically assign me as the mentor of the member I added so that accountability is clear.
* As a Mentor, I want to view the full details of a member so that I can see their complete discipleship profile.
* As a user, I want to update the progress stage of my follow-up members so that their journey is accurately recorded.
* As a user, I want to see the progress history of a member so that I can track when and how their stage changed.
* As a user, I want to see who changed the progress of a member so that there is accountability in updates.
* As a user, I want to search members by name so that I can quickly find a specific person.
* As a user, I want to filter members by progress stage so that I can see all members at a specific stage.
* As a user, I want to view all mentors in the system so that I can see who is active in discipleship.
* As a user, I want to click on a mentor and see their follow-up members so that I have visibility across the team.
* As a user, I want to see a summary of a mentor's follow-ups by progress stage so that I can quickly assess their pipeline.
* As a user, I want to see a dashboard with stat cards so that I have a quick overview of the discipleship pipeline.
* As a user, I want to see the total number of members so that I know the overall size of the community.
* As a user, I want to see how many members are in each stage so that I can identify where most people are in their journey.
