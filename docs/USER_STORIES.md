# User Stories
## Ito ang format: "As a [user], I want to [action], so that [reason]."
### Admin

* As an Admin, I want to add new members so that they are tracked in the system.
* As an Admin, I want to assign a mentor to each member so that accountability is clear.
* As an Admin, I want to view all members and their progress so that I can see the overall discipleship pipeline.
* As an Admin, I want to update a member's classification so that the records stay accurate.

### Mentor / Cell Leader

* As a Mentor, I want to view my assigned members so that I can focus on who I am responsible for.
* As a Mentor, I want to update a member's progress stage so that it reflects their current discipleship journey.
* As a Mentor, I want to add details/notes about a member so that important life updates are recorded.
