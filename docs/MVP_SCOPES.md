# MVP Scope
## Ito ang pinakamaliit na version ng system na may saysay na gamitin.
### IN — Kasama sa MVP
* Member CRUD - Core ng system
* Progress tracking (dropdown) - Main purpose
* Mentor assignment - Accountability
* Classification tagging - Segmentation
* Details/notes field - Life updates
* Admin & Mentor roles - Basic access control
* Simple dashboard (list view) - Para makita agad ang lahat

### OUT — Hindi muna sa MVP
* Member self-login portal - Complex
* Notifications / reminders - Nice to have
* Analytics / charts - Later phase
* Mobile app - Web muna
* Export to PDF/Excel - Later phase
